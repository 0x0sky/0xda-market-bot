This is the first versioned release of the private 0xda Market Telegram client.

## Highlights

- Passwordless Telegram authentication and role-aware command menus.
- Database-driven localized `/buy` catalog.
- Localized `/apply_prices` and `/apply_price` admin workflows.
- Daily admin price digest with CET/CEST handling.
- Resilient webhook and market API behavior during cold starts.
- Isolated test and production Render/Telegram environments.

## Compatibility note

**Breaking:** `premium_9m` is replaced by `premium_12m`. This bot requires
[`0xda-market` v0.1.0](https://github.com/0x0sky/0xda-market/releases/tag/v0.1.0)
or a compatible core exposing the localized catalog and pricing proposal API.

Promote and verify the core before promoting the bot. No bot-side database
migration is required, and the existing Telegram webhook contract is unchanged.

## Verification after promotion

- `GET /health` returns HTTP `200` and the exact production commit in
  `revision`.
- `/start` authenticates against the production core and installs the correct
  role-specific command scope.
- `/buy` shows nine localized products including `premium_12m`.
- `/apply_prices` is visible only to admins and renders database-defined rows.
- One controlled `/apply_price` update is accepted and reflected in a second
  `/apply_prices` request.
- The production digest service remains scheduled for 07:00 CET/CEST.
